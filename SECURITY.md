# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately rather than
opening a public issue.

- Email: **info@innovayse.com**
- Include a description of the vulnerability, steps to reproduce, and any relevant logs or
  proof-of-concept code.

We'll acknowledge your report within a few business days and keep you updated as we investigate
and address the issue. Please give us a reasonable amount of time to fix the issue before any
public disclosure.

## Supported Versions

This project is under active development on `main`. Security fixes are applied to the latest
version only.
